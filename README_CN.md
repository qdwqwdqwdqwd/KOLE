## 🎯 使命

KOLE 正在构建一个去中心化的监督生态系统，利用区块链技术曝光并永久记录关键意见领袖（KOL）的不当行为。通过透明的社区治理和不可篡改的证据存储，我们保护投资者和消费者免受数字影响力经济中的欺诈行为侵害。

## 🌟 核心功能

### 🔐 **不可篡改的证据存储**
- **基于区块链**：所有证据永久存储在 Solana 区块链上
- **IPFS 集成**：分布式存储确保证据可用性
- **防篡改**：加密哈希防止证据被操纵
- **公开验证**：任何人都可以在链上验证证据真实性

### ⚖️ **去中心化评审系统**
- **社区陪审团**：多评审员共识机制
- **独立评审**：盲审过程防止串通
- **严重程度评级**：5级不当行为分类系统
- **一致批准**：所有评审员必须同意才能认证

### 💰 **代币经济（KOLE）**
- **总供应量**：1,000,000,000 KOLE（固定）
- **奖励系统**：激励证据提交和评审
- **幸运抽奖**：为代币持有者提供每小时和每日奖励
- **通缩机制**：定期回购和销毁机制

### 🏛️ **DAO 治理**
- **社区驱动**：代币持有者对平台决策进行投票
- **透明**：所有治理流程都在链上
- **渐进式去中心化**：逐步过渡到完全 DAO 控制

## 🚀 快速开始

### 前置要求
- Node.js 16+
- Solana CLI 工具
- Git

### 安装步骤
```bash
# 克隆仓库
git clone https://github.com/qdwqwdqwdqwd/KOLE.git
cd KOLE

# 安装依赖
npm install

# 设置环境变量
cp .env.example .env
# 编辑 .env 配置文件

# 运行开发服务器
npm run dev
```
